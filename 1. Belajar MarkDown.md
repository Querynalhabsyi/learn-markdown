# Belajar Markdown

## Headers

Untuk membuat headers, dibutuhkan hashtag (#). 
Semakin banyak #, semakin kecil tulisannya.
`#` Lightning.
**#** Bold.
_Test_ Italic.
__Test__ Bold

_Test_
Pakai shift supaya tidak underscore 2x.
_Test_
